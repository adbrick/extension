# extension

The AdBrick extension

## Build Instructions

- Install dependencies: `make`, `node` (any modern version)
- Run: `make build`
- Unpacked locations:
  - Chromium: `dist/chromium`
  - Firefox: `dist/firefox`
- Packed locations (zip files):
  - Chromium: `dist/chromium.zip`
  - Firefox: `dist/firefox.zip`
